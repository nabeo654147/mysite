## Requirement

- next.js 12.0.7

## Installation

開発の始めには以下のようにインストールしてください。<br>
今回は`yarn`を使用していますが、`npm`でも大丈夫です。

```bash
## GitHubからのクローンの後...
cd mysite
yarn install
```

## Usage

```bash
yarn dev
```
